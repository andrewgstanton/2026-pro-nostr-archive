# Note 3 — Sovereignty Is a UX Constraint - 2/5/2026

If people can’t understand or control it, it isn’t sovereign.

Sovereignty that requires a philosophy degree is dead on arrival.

Good sovereign systems:
- default to clarity  
- expose assumptions  
- fail loudly  
- don’t moralize

Freedom should feel *grounded*, not ceremonial.
