The note was already created and signed before the network was involved.



What this example shows

→ write offline  
→ sign locally  
→ publish later  
→ network as distribution only  



Key takeaway

Authorship, identity, and signing DO NOT require the internet.

The network is optional for creation — only required for distribution.



This setup requires running Continuum locally.

That’s the tradeoff for independence from connectivity.
