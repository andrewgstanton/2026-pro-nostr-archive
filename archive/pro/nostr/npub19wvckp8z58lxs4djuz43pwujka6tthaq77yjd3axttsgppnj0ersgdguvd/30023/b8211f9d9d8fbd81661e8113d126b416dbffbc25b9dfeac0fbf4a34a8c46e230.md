- a file format
- a pipeline
- a repeatable system

**Publishing should be owned, repeatable, and local-first — not dependent on external platforms.**

- source → structured markdown  
- metadata → YAML  
- scripts → EPUB output  
- validation → preview tools

That is the important part.
