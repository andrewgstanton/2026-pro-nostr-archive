Steps

## 1. clone to "continuum-pro-test"

```
git clone git@github.com:andrewgstanton/continuum-pro.git continnum-pro-test
```

## 2. checkout latest branch

```
git checkout latest
```

## 3. run build (to generate image)

```
cd dashboard
chmod +x *.sh
./clean_all.sh
./clean.sh
./build.sh
```

## 4. remove source and use run script tailored for distribution

```
./mk_prod.sh
```

## 5. zip directory and produce shahash from it

```
zip -r continuum-pro-test.zip continuum-pro-test/ \
  -x "*.DS_Store" \
  -x "__MACOSX/*" \
  -x "*/.DS_Store" \
  -x "*/._*" \
  -x ".Spotlight-V100/*" \
  -x ".Trashes/*"

unzip -t continuum-pro-test.zip (should say 'No errors detected in compressed data of continuum-pro-test.zip.')

shasum -a 256 continuum-pro-test.zip > continuum-pro-test.zip.sha256.txt

```

## 6. copy these files to the download folder:

```
continuum-pro-test.zip
continuum-pro-test.zip.sha256.txt

(from the continuum-pro-test directory root)

continuum-overview.pdf
continuum-overview.md
README.md
README_MAC.md
README_WINDOWS.md
LICENSE.txt
NOTICE.md

```
